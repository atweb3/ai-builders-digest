# 🤖 AI Builders Digest

每天自动推送 AI 领域最新干货到飞书。

## ✨ 效果
- 每天 08:30 和 16:30 自动推送
- 内容来自 25+ 位 AI Builder
- 推送格式：飞书消息

## 🚀 怎么用？
1. 点右上角 Star ⭐ 支持一下
2. 查看 [Actions](https://github.com/atweb3/ai-builders-digest/actions) 运行记录
3. 加入飞书群接收推送

## 📦 技术栈
- GitHub Actions
- Node.js
- 飞书 Webhook
